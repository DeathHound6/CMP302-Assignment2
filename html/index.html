About The Site

Welcome to our library web system. This site will allow you to search for various books, open them as an ebook or created a plan to borrow a book for x amount of days.

All physical books that are borrowed from this library need to be returned within the plan that has been set by the user and agreed by the service. All physical books need to be returned in the same quality with which they were borrwed. Books that have been damaged when in the holders possession will need to be substituted with cash payment of the books value.
